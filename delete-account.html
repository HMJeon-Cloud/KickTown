계정 삭제 안내

서비스명: 동네 축구왕(KickTown) · 앱 내에서 직접 탈퇴할 수 있습니다.

1. 앱에서 직접 탈퇴하기 (권장)

  1. 앱에 로그인합니다.
  2. 하단 [더보기] 탭으로 이동합니다.
  3. [계정] 항목에서 [회원탈퇴]를 선택합니다.
  4. 삭제되는 정보와 남는 정보 안내를 확인합니다.
  5. 본인 확인을 진행합니다.
    • 이메일 계정: 현재 비밀번호 입력
    • 카카오 계정: 확인 문구 "탈퇴" 입력
  6. 동의 항목에 체크한 뒤 [탈퇴하기]를 누르고, 마지막 확인 창에서 확인을 선택합니다.

탈퇴는 즉시 처리되며, 완료 후 자동으로 로그아웃됩니다.

2. 이메일로 요청하기

앱에 로그인할 수 없는 등 직접 탈퇴가 어려운 경우, 아래 주소로 요청해 주세요. 가입에 사용한 이메일(또는 카카오 로그인 여부)과 팀 이름을 함께 알려주시면 본인 확인 후 7일 이내에 처리합니다.

문의: (운영자 이메일 주소 입력)

3. 삭제되는 정보와 남는 정보

구분내용
삭제 로그인 계정 정보(이메일 계정 또는 카카오 연결 정보), 생일·키·몸무게 등 프로필 정보, 계정과 팀 명단 사이의 연결
보존 팀의 경기 결과, 득점·도움, 출석 기록 등 팀 운영 이력. 팀 통계의 일관성을 위해 이름 표기만 남으며 계정과의 연결은 끊어집니다.

이름 표기까지 완전히 삭제하기를 원하시면, 위 이메일로 별도 요청해 주세요.

4. 유의사항

  • 탈퇴 후에는 되돌릴 수 없으며, 동일한 계정으로 다시 로그인할 수 없습니다.
  • 구단주로 등록된 팀이 있는 경우, 탈퇴하면 해당 팀은 구단주가 없는 상태가 됩니다. 팀을 계속 운영하려면 탈퇴 전에 다른 임원에게 권한을 넘기시길 권장합니다.
  • 팀 자체의 데이터(경기·명단·게시글)는 탈퇴로 삭제되지 않습니다.
